Get the Mailvelope extension from Chrome

There are several email clients like Enigmail, Mutt and Evolution that can be used to encrypt and sign emails with PGP keys. This help is going to talk about using the Chrome extension, Mailvelope to encrypt and sign email easily.

To get the plugin

  1. Select  ▸ Tools ▸ Extensions.

  2. Click the Get more extensions link.

  3. In the Chrome Web Store search for the Mailvelope extension.

  4. Click the ADD TO CHROME button and confirm your decision by clicking Add in the confirmation dialog.

  5. There will be small button on the top right side of your chrome browser if your installation is successful.
