Make your keys readable

Export your public and private keys to text files. The contents of these text files can then be copied into Mailvelope while importing keys.

Export your private keys

General Syntax to export key

  $
  gpg --export-secret-key -a <your key id> > < location of your key file>

An example using the listed key

  $
  gpg --export-secret-key -a 7761B227 > ~/lupin-private.key

Your private key will be present in the lupin-private.key file in your Home(~) directory. To look at the contents of the key file use an editor like gedit, so type the following command in the Terminal. Alternatively, you can right click the key file and select Open or double click the key file to open it.

gedit lupin-private.key

Export your public keys

General Syntax to export key

  $
  gpg --export-key -a <your key id> > < location of your key file>

An example using the listed key

  $
  gpg --export-secret-key -a 7761B227 > ~/lupin-public.key

Your public key will be present in the lupin-public.key file in your Home(~) directory. To look at the contents of the file, type the following command in the terminal. Alternatively, you can right click the key file and select Open or double click the key file to open it.

gedit lupin-public.key
