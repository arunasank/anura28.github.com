Get the GPG tarball

Use this page only if GPG is not installed on your computer. To check if you have GPG, open your Terminal and type $gpg --version. This will give you the GPG version if GPG has been installed or ask you to install GPG.

GPG is available for free download on the official GnuPG website

Screenshot of exact download link

Only download the tarball for GnuPG 2.0.20 full source code

By default, the tarball, gnupg-2.0.20.tar.bz2 will be saved in your ~/Downloads folder unless you have chosen some other folder to store the downloaded tarball.
