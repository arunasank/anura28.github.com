Install GPG on your computer

To install GPG:

  1. Use Nautilus to go to the folder which contains the gnupg-2.0.20.tar.bz2 tarball.

  2. Select the gnupg-2.0.20.tar.bz2 tarball. Right click on the tarball and select Extract Here.Now, there will be a folder called gnupg-2.0.20 in the current folder.

  3. To build from source, open your terminal:

            $ cd <directory which contains the extracted folder>
	$ cd gnupg-2.0.20
        $gnupg-2.0.20/ ./configure
        Output after configure.
        $gnupg-2.0.20/ make
        Output after make.
        $gnupg-2.0.20/ sudo make install 
        Output after make install.

  4. Check if GnuPG 2.0 has been installed.

    An output similar to the following shows that GnuPG 2.0.20 has been installed

              $ gpg2 --version
